# 行为准则

## 我们的承诺

为了营造一个开放、包容的环境，我们作为贡献者和维护者承诺：
无论年龄、体型、残疾、种族、性别特征、性别认同和表达、经验水平、教育、社会经济地位、国籍、外表、种族、宗教或性取向如何，
参与本项目和维护项目的人都不会受到骚扰。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 专注于对社区最有利的事情
- 对其他社区成员表示同情

不可接受的行为包括：

- 使用与性相关的语言或图像，以及不受欢迎的性 attention 或 advances
- 侮辱、贬损的评论，人身或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息
- 其他在专业环境中可被合理认为不恰当的行为

## 我们的责任

项目维护者有责任解释可接受行为的标准，
并对任何不可容忍的行为采取适当和公正的纠正措施。

项目维护者有权和责任删除、编辑或拒绝评论、提交、代码、wiki 编辑、issue 和其他不符合本行为准则的贡献，
并在适当时解释 moderation 决定。

## 范围

本行为准则适用于项目空间和公共空间，
当个人代表项目或社区时。

## 执行

可以通过 apples398@163.com 联系项目团队报告虐待、骚扰或其他不可接受的行为。
所有投诉将被审查和调查，并在必要时作出回应。
项目团队对 reporter 的机密性负有义务。

不遵循或执行行为准则的维护者可能面临由其他项目领导成员决定的暂时或永久禁令。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 版本 2.1。
